## 2026-06-14

### 创建 MD3 风格静态下载站框架
- 搭建完整文件结构：根 index.html（引导跳转）、page/（站点主体）、file/publish1/（示例项目）
- 实现 Material Design 3 全套 CSS 设计令牌（亮色/暗色主题），支持主题切换并持久化 localStorage
- 核心功能：搜索筛选、分类 Filter Chips、卡片网格、侧边栏详情面板（Side Sheet）、涟漪动画、Snackbar Toast
- 自动平台检测，推荐当前系统对应的版本
- 项目数据通过 `file/<project>/main.json` 定义，`page/config.json` 注册项目列表
- 纯静态，无构建工具，可直接部署
